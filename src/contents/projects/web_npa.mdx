## Short Explanation

Sistem informasi ini dibuat untuk mengatasi beberapa persoalan yang dihadapi perusahaan. Persoalan pertama yaitu internal operasional perusahaan ingin data network performance area khususnya NPA Area 3 yaitu Jateng-DIY, Jatim, dan Balinusra. Visualisasi data dapat dilihat baik pada browser / website maupun melalui aplikasi mobile. Perusahaan juga ingin data dengan volume besar (big data) dapat diolah dan ditampilkan secara efektif dan efisien sehingga timbul persoalan pemilihan sistem database. Selain itu, diperlukan juga penghubung antara aplikasi yang menampilkan data dengan sumber data atau database. Untuk menghubungkan kedua hal tersebut, dibutuhkan antarmuka pemrograman atau Application Programming Interface (API).
